# 🚆 Train Ticket Booking System

A simple and responsive **Train Ticket Booking Web Application** built using **HTML, CSS, JavaScript, and JSON (db.json)**.

This project simulates a real-world railway booking system where users can select trains, check seat availability, and book tickets using a local database.

---

## 🌟 Features

* 🔍 Train selection system
* 🪑 Seat availability display
* 🎫 Ticket booking system
* 📦 Local database using db.json
* 💳 Fake payment animation
* 📱 Fully responsive design

## ⚙️ How to Run Locally

⚠️ This project uses **JSON Server**, so follow these steps:

### 1. Install JSON Server

npm install -g json-server

### 2. Start Server

json-server --watch db.json

### 3. Open Project

* Open `index.html` in browser
* API runs at: http://localhost:3000
